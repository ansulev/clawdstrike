**2. `apps/workbench/src/features/activity-bar/stores/activity-bar-store.ts`**

Create a Zustand store using `create` + `immer` middleware + `createSelectors` wrapper:

```typescript
import { create } from "zustand";
import { immer } from "zustand/middleware/immer";
import { createSelectors } from "@/lib/create-selectors";
import type { ActivityBarItemId } from "../types";

interface ActivityBarState {
  activeItem: ActivityBarItemId;
  sidebarVisible: boolean;
  sidebarWidth: number;
  actions: {
    setActiveItem: (id: ActivityBarItemId) => void;
    toggleItem: (id: ActivityBarItemId) => void;
    toggleSidebar: () => void;
    showPanel: (id: ActivityBarItemId) => void;
    setSidebarWidth: (width: number) => void;
    collapseSidebar: () => void;
  };
}
```

Default state: `activeItem: "explorer"`, `sidebarVisible: true`, `sidebarWidth: 240`.

Action implementations:
- `setActiveItem(id)`: sets `activeItem` to `id`
- `toggleItem(id)`: if `id === state.activeItem && state.sidebarVisible`, set `sidebarVisible: false`. Otherwise set `activeItem: id, sidebarVisible: true`.
- `toggleSidebar()`: flip `sidebarVisible`
- `showPanel(id)`: set `activeItem: id, sidebarVisible: true`
- `setSidebarWidth(width)`: set `sidebarWidth` to `Math.max(120, Math.min(480, width))`
- `collapseSidebar()`: set `sidebarVisible: false`

Export as `export const useActivityBarStore = createSelectors(create<ActivityBarState>()(immer((set) => ({ ... }))));`
