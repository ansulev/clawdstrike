       - `private handlers: Map<string, BridgeHandler>` -- maps method names to handler functions
       - `private disposables: (() => void)[]` -- tracks registered contributions for cleanup
       - `private storage: Map<string, unknown>` -- plugin-scoped key/value store
       - `private pluginId: string`
       - `private targetWindow: Window`
       - `private allowedOrigin: string`

       Constructor:
       - Stores options, defaults allowedOrigin to "null"
       - Calls `this.registerDefaultHandlers()` to wire up all 7 PluginContext methods

       `registerDefaultHandlers()`:
       - "commands.register" -> stores command metadata, returns { registered: true }. Note: the command handler itself stays in the iframe. The host stores the command contribution data and when the command is invoked by the host, it will send a "commands.invoke" event to the iframe (the invoke event pattern is future work for Phase 2+, but the registration is needed now).
       - "guards.register" -> calls `registerGuard(params)`, pushes dispose, returns { registered: true }
       - "fileTypes.register" -> calls `registerFileType(params)`, pushes dispose, returns { registered: true }
       - "statusBar.register" -> calls `statusBarRegistry.register(...)`, pushes dispose, returns { registered: true }
       - "sidebar.register" -> stores contribution data, returns { registered: true }
       - "storage.get" -> returns `this.storage.get(params.key)` (params is `{ key: string }`)
       - "storage.set" -> `this.storage.set(params.key, params.value)`, returns undefined

       `handleMessage(event: MessageEvent)`:
       - Check `event.origin === this.allowedOrigin`, silently drop if mismatch
       - Check `isBridgeMessage(event.data)` and `event.data.type === "request"`, ignore otherwise
       - Look up handler for `event.data.method`
       - If no handler: reply with BridgeErrorResponse { code: "METHOD_NOT_FOUND" }
       - If handler exists: try/catch, on success send BridgeResponse, on error send BridgeErrorResponse { code: "INTERNAL_ERROR" }
       - All replies sent via `this.targetWindow.postMessage(reply, "*")`

       `pushEvent(method: string, params?: unknown)`:
       - Sends `{ type: "event", method, params }` to targetWindow

       `registerHandler(method: string, handler: BridgeHandler)`:
       - Allows external code to register additional bridge handlers (for extensibility)

       `destroy()`:
       - Calls all disposables
       - Clears handlers, storage, disposables
