    Create `packages/sdk/plugin-sdk/src/testing.ts`:

    1. Import all types from `./types` and all context interfaces from `./context` (use `import type` for types).

    2. **MockStorageApi class** implementing StorageApi:
       - Private `_store: Map<string, unknown>`
       - `get(key)`: returns `this._store.get(key)`
       - `set(key, value)`: calls `this._store.set(key, value)`
       - `entries()`: returns `Array.from(this._store.entries())`
       - `clear()`: convenience method to reset state

    3. **MockSecretsApi class** implementing SecretsApi:
       - Private `_store: Map<string, string>`
       - `get(key)`: returns `Promise.resolve(this._store.get(key) ?? null)`
       - `set(key, value)`: sets and returns `Promise.resolve()`
       - `delete(key)`: deletes and returns `Promise.resolve()`
       - `has(key)`: returns `Promise.resolve(this._store.has(key))`

    4. **SpyContext type** exported as:
       ```typescript
       export interface SpyContext extends PluginContext {
         spy: {
           commands: { registered: Array<{ contribution: CommandContribution; handler: () => void }> };
           guards: { registered: GuardContribution[] };
           fileTypes: { registered: FileTypeContribution[] };
           statusBar: { registered: StatusBarItemContribution[] };
           sidebar: { registered: ActivityBarItemContribution[] };
           views: {
             editorTabs: EditorTabViewContribution[];
             bottomPanelTabs: BottomPanelTabViewContribution[];
             rightSidebarPanels: RightSidebarPanelViewContribution[];
             statusBarWidgets: StatusBarWidgetViewContribution[];
           };
           enrichmentRenderers: { registered: Array<{ type: string; component: ComponentType }> };
           storage: MockStorageApi;
           secrets: MockSecretsApi;
           subscriptions: Disposable[];
         };
       }
       ```

    5. **createMockContext(overrides?: Partial<PluginContext>)** function:
       - Creates a full PluginContext with no-op stubs for every API method
       - storage uses `new MockStorageApi()`
       - secrets uses `new MockSecretsApi()`
       - Spreads `overrides` last to allow selective replacement
       - Returns `PluginContext`

    6. **createSpyContext(overrides?: Partial<PluginContext>)** function:
       - Creates arrays for each tracking category
       - Each `register()` method pushes to its array and returns a Disposable that splices the item out
       - Attaches a `spy` property to the returned context
       - storage and secrets are MockStorageApi/MockSecretsApi instances (also accessible via `spy.storage` and `spy.secrets`)
       - Spreads `overrides` last (overridden APIs won't have spy tracking -- that's expected)
       - Returns `SpyContext`
