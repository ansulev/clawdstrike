<objective>
Create the plugin view tab state management store and keep-alive renderer for plugin editor tabs.

Purpose: Plugins can open full-panel views in the editor area as tabs. The tab store tracks which plugin views are open, which is active, and handles LRU eviction. The renderer keeps inactive tabs mounted (display:none) so component state (scroll, forms, selections) survives tab switches.

Output: PluginViewTabStore with open/close/activate/LRU + ViewTabRenderer with keep-alive pattern
</objective>

Define PluginViewTab interface:
```typescript
export interface PluginViewTab {
  /** Same as the ViewRegistration id: "{pluginId}.{viewId}" */
  viewId: string;
  /** Display label (initially from ViewRegistration.label, updatable via setTitle) */
  label: string;
  /** Icon identifier (from ViewRegistration.icon) */
  icon?: string;
  /** Whether this tab has unsaved changes (set via setDirty) */
  dirty: boolean;
  /** Timestamp when this tab was opened (for ordering) */
  openedAt: number;
  /** Timestamp of last activation (for LRU eviction) */
  lastActiveAt: number;
}
```

Module-level state:
- `const tabMap = new Map<string, PluginViewTab>()`
- `let activeTabId: string | null = null`
- `const listeners = new Set<() => void>()`
- `let tabsSnapshot: PluginViewTab[] = []`
- `let activeIdSnapshot: string | null = null`
- `const MAX_KEPT_ALIVE = 5`

On openPluginViewTab(viewId):
1. If already in tabMap, just call activatePluginViewTab(viewId) and return
2. Look up ViewRegistration via `getView(viewId)` from view-registry.ts. If not found, throw `Error("View \"{viewId}\" not registered")`
3. Create PluginViewTab with label from registration.label, icon from registration.icon, dirty=false, openedAt=Date.now(), lastActiveAt=Date.now()
4. Add to tabMap
5. Set activeTabId = viewId
6. Rebuild snapshots and notify

On activatePluginViewTab(viewId):
1. If viewId is null, set activeTabId = null, rebuild, notify, return
2. If viewId not in tabMap, throw
3. Update tab's lastActiveAt = Date.now()
4. Set activeTabId = viewId
5. Run LRU eviction: count entries where viewId !== activeTabId. While count > MAX_KEPT_ALIVE, find entry with smallest lastActiveAt among non-active entries, delete it from tabMap
6. Rebuild snapshots and notify

On closePluginViewTab(viewId):
1. Delete from tabMap
2. If activeTabId was the closed tab, find the remaining tab with highest lastActiveAt, set activeTabId to its viewId (or null if tabMap is empty)
3. Rebuild snapshots and notify

rebuildSnapshots: set tabsSnapshot to Array.from(tabMap.values()) sorted by openedAt ascending. Set activeIdSnapshot = activeTabId.

Export two separate useSyncExternalStore hooks:
- usePluginViewTabs: subscribe=onPluginViewTabChange, getSnapshot=getOpenPluginViewTabs
- useActivePluginViewTabId: subscribe=onPluginViewTabChange, getSnapshot=getActivePluginViewTabId

For getOpenPluginViewTabs, return a frozen empty array when tabMap is empty (same pattern as view-registry).

Also export setPluginViewTabTitle(viewId, title) and setPluginViewTabDirty(viewId, dirty) which update the respective tab fields and rebuild+notify.

```tsx
function PluginEditorTabBridge({ registration, viewId, isActive }: { ... }) {
  const handleSetTitle = useCallback((title: string) => {
    setPluginViewTabTitle(viewId, title);
  }, [viewId]);
  const handleSetDirty = useCallback((dirty: boolean) => {
    setPluginViewTabDirty(viewId, dirty);
  }, [viewId]);

  const BridgedComponent = useMemo(() => {
    const Original = registration.component;
    // Create a component that receives ViewProps from ViewContainer
    // and adds EditorTabProps
    function EditorTabBridge(props: any) {
      return <Original {...props} setTitle={handleSetTitle} setDirty={handleSetDirty} />;
    }
    EditorTabBridge.displayName = `EditorTabBridge(${viewId})`;
    return EditorTabBridge;
  }, [registration.component, viewId, handleSetTitle, handleSetDirty]);

  // Create a modified registration with the bridged component
  const bridgedRegistration = useMemo(() => ({
    ...registration,
    component: BridgedComponent,
  }), [registration, BridgedComponent]);

  return (
    <ViewContainer
      registration={bridgedRegistration}
      isActive={isActive}
      slotType="editorTab"
    />
  );
}
```
