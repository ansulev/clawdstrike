Create `apps/workbench/src/lib/plugins/plugin-registry.ts`:

Use a class-based singleton pattern (same as Athas ExtensionRegistry, same as existing command-registry pattern in the codebase).

```
class PluginRegistry {
  private plugins: Map<string, RegisteredPlugin>;
  private listeners: Map<PluginRegistryEventType, Set<PluginRegistryCallback>>;
}
```

1. **Event types:**
   - `type PluginRegistryEventType = "registered" | "unregistered" | "stateChanged"`
   - `interface PluginRegistryEvent { type: PluginRegistryEventType; pluginId: string; plugin?: RegisteredPlugin; oldState?: PluginLifecycleState; newState?: PluginLifecycleState; }`

2. **PluginRegistrationError** class extending Error:
   - `validationErrors?: ManifestValidationError[]`
   - Constructor takes message + optional validationErrors array

3. **register(manifest: PluginManifest): void**
   - Call `validateManifest(manifest)` -- if invalid, throw `PluginRegistrationError` with the validation errors
   - If `plugins.has(manifest.id)` already, throw `PluginRegistrationError("Plugin '${manifest.id}' is already registered")`
   - Create `RegisteredPlugin` with state `"installed"`, `installedAt: Date.now()`
   - Store in Map
   - Emit "registered" event

4. **unregister(id: string): void**
   - If not found, return silently (no-op)
   - Delete from Map
   - Emit "unregistered" event

5. **get(id: string): RegisteredPlugin | undefined**
   - Return `plugins.get(id)`

6. **getAll(): RegisteredPlugin[]**
   - Return `Array.from(plugins.values())`

7. **getByContributionType(type: ContributionPointType): RegisteredPlugin[]**
   - Filter all plugins where `plugin.manifest.contributions?.[type]` is a non-empty array
   - Return matching plugins

8. **setState(id: string, state: PluginLifecycleState, error?: string): void**
   - Get plugin from Map, throw if not found
   - Store old state
   - Update `plugin.state = state`
   - If state is "activated", set `plugin.activatedAt = Date.now()`
   - If state is "error" and error is provided, set `plugin.error = error`
   - Emit "stateChanged" event with oldState and newState

9. **subscribe(eventType: PluginRegistryEventType, callback: (event: PluginRegistryEvent) => void): () => void**
   - Add callback to the listeners Set for that eventType
   - Return dispose function that removes the callback

10. **reset(): void** (for tests and hot reload)
    - For each plugin, emit "unregistered"
    - Clear the Map

11. Private **emit(event: PluginRegistryEvent): void**
    - Get listeners for event.type, call each

Export singleton: `export const pluginRegistry = new PluginRegistry()`
Export the class too: `export { PluginRegistry }` (for testing with fresh instances)
