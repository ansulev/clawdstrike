Create `bottom-panel-tabs.tsx` with a `BottomPanelTabs` component that:

1. Defines a `BuiltInTab` interface: `{ id: string; label: string; icon: ComponentType; content: ReactNode }`.

2. Accepts props:
   - `builtInTabs: BuiltInTab[]` -- the caller provides built-in tab definitions (Problems, Test Runner, etc.) so the component is not coupled to specific built-in panels.
   - `panelHeight: number` -- current height of the bottom panel area for passing to plugin views.
   - `activeTabId: string | null` -- which tab is selected.
   - `onTabChange: (tabId: string) => void` -- callback when user clicks a tab.

3. Calls `useViewsBySlot("bottomPanelTab")` to get plugin tabs from the ViewRegistry.

4. Merges built-in tabs and plugin tabs into a unified tab bar. Built-in tabs appear first (in the order given), then plugin tabs sorted by priority. Each tab shows its icon (if any) and label.

5. Tab bar styling: horizontal row of tabs at the top of the bottom panel. Use the same workbench style conventions: `bg-[#0b0d13]` background, `text-[#6f7f9a]` inactive text, `text-[#ece7dc]` active text, `border-b-2 border-[#d4a84b]` for active tab indicator. Text size `text-[9px] font-mono`.

6. Content area: renders the active tab's content:
   - For built-in tabs: renders `builtInTabs[activeIndex].content` (the ReactNode passed in).
   - For plugin tabs: renders `<ViewContainer registration={pluginRegistration} isActive={true} />` and passes `panelHeight` as an additional prop by wrapping the plugin component to inject it.

7. To pass `panelHeight` to plugin components: create a thin wrapper component that spreads `panelHeight` into the ViewContainer's rendered component. Since ViewContainer uses `ViewProps` (viewId, isActive, storage) and `BottomPanelTabProps` extends ViewProps with `panelHeight`, create the wrapper as:
   ```typescript
   function BottomPanelPluginView({ registration, panelHeight }: { registration: ViewRegistration; panelHeight: number }) {
     // Clone registration with a wrapped component that injects panelHeight
     const wrappedRegistration = useMemo(() => ({
       ...registration,
       component: (props: ViewProps) => {
         const Component = registration.component;
         return <Component {...props} panelHeight={panelHeight} />;
       },
     }), [registration, panelHeight]);
     return <ViewContainer registration={wrappedRegistration} isActive={true} />;
   }
   ```

8. If no tabs exist at all (no built-in and no plugins), render nothing. If the active tab ID does not match any tab, default to the first available tab.

Create tests in `__tests__/bottom-panel-tabs.test.tsx` that mock `useViewsBySlot` to return test plugin registrations and verify the 6 behaviors listed above. Mock ViewContainer as a simple div that displays viewId.
