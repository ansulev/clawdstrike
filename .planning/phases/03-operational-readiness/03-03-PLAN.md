Build the plugin API key settings UI and finding list enrichment badges, providing the operator-facing configuration and status visibility surfaces for the threat intel plugin ecosystem.

Output: A new "Plugins" tab in Settings with generic API key forms, and enrichment source badges on finding list rows.
</objective>
