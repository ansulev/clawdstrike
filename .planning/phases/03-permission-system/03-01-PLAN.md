<objective>
Define the permission type system and enforce capability-based permissions at the bridge middleware layer.

Purpose: Community plugins must declare the capabilities they need. The bridge host rejects any API call the plugin did not declare permission for -- fail-closed. This is the Chrome extension model adapted for ClawdStrike's postMessage bridge.

Output: PluginPermission type on PluginManifest, METHOD_TO_PERMISSION mapping, permission enforcement middleware in PluginBridgeHost, full TDD test coverage.
</objective>

       // After origin validation, before handler lookup:
