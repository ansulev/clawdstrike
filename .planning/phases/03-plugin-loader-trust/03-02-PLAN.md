    GREEN: Create plugin-loader.ts with:

    ```typescript
    export interface PluginModule {
      activate(context: PluginActivationContext): Disposable[] | void;
      deactivate?(): void;
    }

    export interface PluginActivationContext {
      pluginId: string;
      subscriptions: Disposable[];
    }

    export type Disposable = () => void;

    export type ModuleResolver = (manifest: PluginManifest) => Promise<PluginModule>;

    export interface PluginLoaderOptions {
      registry?: PluginRegistry;
      resolveModule?: ModuleResolver;
      trustOptions?: TrustVerificationOptions;
    }
    ```

    PluginLoader class (exported as class + singleton):
    - `private loadedPlugins = new Map<string, { disposables: Disposable[]; module: PluginModule }>()`
    - `private pendingActivation = new Map<string, PluginManifest>()` -- plugins waiting for activation events

    constructor(options?: PluginLoaderOptions):
    - Stores registry (defaults to pluginRegistry singleton)
    - Stores resolveModule (defaults to dynamic import: `async (m) => import(m.main!)`)
    - Stores trustOptions (defaults to `{}`)

    async loadAll(): Promise<void>:
    - Gets all plugins from registry in "installed" state
    - Splits into startupPlugins (shouldActivateOnStartup) and deferredPlugins
    - For deferredPlugins, stores them in pendingActivation map
    - For startupPlugins, calls Promise.allSettled(startupPlugins.map(p => this.loadPlugin(p.manifest.id)))
    - Logs settled results (fulfilled/rejected)

    async loadPlugin(pluginId: string): Promise<void>:
    - Gets plugin from registry, throws if not found
    - 1. Trust gate: call verifyPluginTrust(manifest, this.trustOptions). If not trusted, set state to "error" with reason, return early.
    - 2. Set state to "activating"
    - 3. Resolve module: call this.resolveModule(manifest)
    - 4. Route contributions BEFORE calling activate():
      - For each manifest.contributions.guards: call registerGuard() with a GuardMeta built from the GuardContribution fields, store dispose function
      - For each manifest.contributions.fileTypes: call registerFileType() with a FileTypeRegistrationOptions built from the FileTypeContribution fields, store dispose function
      - For each manifest.contributions.statusBarItems: call statusBarRegistry.register() with a StatusBarItem built from the StatusBarItemContribution fields, store dispose function
    - 5. Create PluginActivationContext with pluginId and empty subscriptions array
    - 6. Call module.activate(context), collect returned disposables (if any) + context.subscriptions
    - 7. Store all disposables (contribution disposables + activate disposables + context.subscriptions)
    - 8. Set state to "activated"
    - On any error: set state to "error" with error message, clean up any already-registered contributions

    async triggerActivationEvent(event: string): Promise<void>:
    - Iterates pendingActivation map
    - For each pending plugin, checks matchActivationEvent(manifest.activationEvents, event)
    - If match: removes from pendingActivation, calls loadPlugin()
    - Uses Promise.allSettled for multiple matches

    async deactivatePlugin(pluginId: string): Promise<void>:
    - Gets loaded plugin data from loadedPlugins map
    - Calls module.deactivate() if it exists
    - Calls all stored disposables (unregisters contributions)
    - Removes from loadedPlugins map
    - Sets registry state to "deactivated"

    Import PluginManifest from "./types".
