  <action>
Create `apps/workbench/src/features/navigation/quick-open-dialog.tsx` exporting a `QuickOpenDialog` component and a Zustand-based open/close mechanism.

**Store state (module-level, no separate store file needed):**
- `_quickOpenVisible: boolean` controlled by exported `openQuickOpen()` and `closeQuickOpen()` functions, plus a `useQuickOpenVisible()` hook that subscribes via `useSyncExternalStore`.

**QuickOpenDialog component:**
- Modal overlay matching command-palette.tsx styling: `fixed inset-0 z-50` backdrop, centered dialog at `top-[15vh]` with `max-w-[560px]`, same bg/border/text color tokens.
- Input field at top with placeholder "Search files by name..." and auto-focus on open.
- File list below input, max-height scrollable.

**Fuzzy matching logic:**
- Write a `flattenProjectFiles(files: ProjectFile[]): ProjectFile[]` helper that recursively collects all non-directory entries from the project file tree.
- Fuzzy match: split query into characters, check if all characters appear in order in the file name (case-insensitive). Score by: consecutive character matches get bonus, match at start gets bonus, shorter file names rank higher. Keep it simple -- no external library needed.
- When query is empty, show recent files from `getRecentFiles()` (imported from policy-store.tsx) mapped to `{ path, name }` objects. Cap at 10 entries. Show a "Recent Files" section header when displaying recent files.

**File selection:**
- On Enter or click, read the file from disk using Tauri `fs.readTextFile` (import from `@tauri-apps/plugin-fs`). Then call `usePolicyTabsStore.getState().openTabOrSwitch(filePath, fileType, content, fileName)` where fileType is determined from the ProjectFile entry. Also call `pushRecentFile(filePath)` from policy-tabs-store.
- Close the dialog after selection.

**Keyboard navigation:**
- ArrowUp/ArrowDown to move selection, Enter to confirm, Escape to close.
- Match the command-palette.tsx pattern: `selectedIndex` state, `data-active` attribute, scroll into view.

**Color tokens (match existing):**
- Background: `bg-[#0b0d13]`
- Border: `border-[#2d3240]`
- Text: `text-[#ece7dc]`
- Muted text: `text-[#6f7f9a]`
- Active highlight: `bg-[#131721]/60` with `border-left: 3px solid #d4a84b`
- Input caret: `caret-[#d4a84b]`

**Integration point:** Export `QuickOpenDialog`, `openQuickOpen`, `closeQuickOpen`, and `useQuickOpenVisible`.

Note: The `readTextFile` import path is `import { readTextFile } from "@tauri-apps/plugin-fs"`. If this import pattern differs from what exists in the codebase, check existing Tauri fs imports with `grep -r "tauri-apps.*fs" apps/workbench/src/ | head -5` and match the pattern found.
