    After the last existing union member (`SET_SESSION_METADATA`), add 3 new members:
    ```typescript
    | { type: "TOPOLOGY_LAYOUT"; topology: string; positions: Map<string, { x: number; y: number }> }
    | { type: "ENGINE_SYNC"; engineNodes: Array<{ id: string; agentId?: string; taskId?: string; data: Partial<SwarmBoardNodeData>; position?: { x: number; y: number } }>; engineEdges: SwarmBoardEdge[] }

    ```
    Use `string` for topology/verdict types to avoid importing engine types into the action union (keeps backward compat clean). The bridge hook will pass the correct string values.

    **Step 2: Add 3 new action methods to the actions namespace (after loadFromBundle).**

    `topologyLayout(topology: string, positions: Map<string, { x: number; y: number }>)`:
    - Get current nodes via `get()`
    - Map each node: if `positions.has(n.id)`, update `n.position` to the Map value, else keep original
    - Call `set({ nodes: updated, selectedNode: deriveSelectedNode(updated, get().selectedNodeId) })`
    - Call `schedulePersist({ ...get() })`
    - Follows the existing `setNodes` pattern at line 766

    `engineSync(engineNodes: Array<{ id: string; agentId?: string; taskId?: string; data: Partial<SwarmBoardNodeData>; position?: { x: number; y: number } }>, engineEdges: SwarmBoardEdge[])`:
    - Build a lookup `Map<string, typeof engineNodes[0]>` keyed on `id`
    - For each current node: if it has `data.engineManaged === true` AND is in the lookup, merge the engine data via `{ ...n, data: { ...n.data, ...lookup.data }, position: lookup.position ?? n.position }`
    - For engine nodes NOT already present in current nodes: create new nodes using `createBoardNode` with `engineManaged: true` in data
    - Merge edges: keep existing edges, add engine edges that don't exist (dedup by `id`)
    - Call `set({ nodes, edges, rfEdges: toRfEdges(edges), selectedNode: deriveSelectedNode(nodes, get().selectedNodeId) })`
    - Call `schedulePersist({ ...get() })`

    `guardEvaluate(agentNodeId: string, verdict: string, guardResults: Array<{ guard: string; allowed: boolean; duration_ms?: number }>, signature?: string, publicKey?: string)`:
    - Get current state
    - Find agent node by `agentNodeId`
    - If not found, return (no-op)
    - Create receipt node via `createBoardNode({ nodeType: "receipt", title: \`Guard: \${verdict.toUpperCase()}\`, position: { x: agentNode.position.x, y: agentNode.position.y + 340 }, data: { verdict: verdict as "allow" | "deny" | "warn", guardResults, signature, publicKey, status: "completed", engineManaged: true } })`
    - Add receipt node to nodes array
    - Create receipt edge: `{ id: \`edge-receipt-\${receiptNode.id}-\${agentNodeId}\`, source: agentNodeId, target: receiptNode.id, type: "receipt", label: verdict }`
    - Add edge to edges array
    - Set nodes, edges, rfEdges, selectedNode
    - Call `schedulePersist`
