2. **Add dirty state subscription** in `pane-store.ts`:
   Create a `subscribeToDirtySync` function that is called once at module level (after store creation):
   ```typescript
   // Sync dirty state from policy-tabs-store to pane views
   function subscribeToDirtySync() {
     const { subscribe } = usePolicyTabsStore;
     subscribe(
       (state) => state.tabs,
       (tabs) => {
         const paneState = usePaneStore.getState();
         const allGroups = getAllPaneGroups(paneState.root);
         let changed = false;
         let nextRoot = paneState.root;

         for (const group of allGroups) {
           for (const view of group.views) {
             if (!view.route.startsWith("/file/")) continue;
             const filePath = view.route.slice("/file/".length);
             const tab = tabs.find((t) => t.filePath === filePath);
             const newDirty = tab?.dirty ?? false;
             const newFileType = tab?.fileType;
             if (view.dirty !== newDirty || view.fileType !== newFileType) {
               changed = true;
               nextRoot = replaceNode(nextRoot, group.id, (node) =>
                 node.type === "group"
                   ? {
                       ...node,
                       views: node.views.map((v) =>
                         v.id === view.id
                           ? { ...v, dirty: newDirty, fileType: newFileType }
                           : v,
                       ),
                     }
                   : node,
               );
             }
           }
         }

         if (changed) {
           usePaneStore.setState({ root: nextRoot });
         }
       },
     );
   }
   ```
   IMPORTANT: Use Zustand's `subscribe` with selector (the `subscribe(selector, listener)` overload). Import `usePolicyTabsStore` at the top of the file.
   Call `subscribeToDirtySync()` after the store creation: place it after `export const usePaneStore = create<PaneStore>(...)`.

   NOTE: The `replaceNode` import is already in the file (imported from `./pane-tree`).
