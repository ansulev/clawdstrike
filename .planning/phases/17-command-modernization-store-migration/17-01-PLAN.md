Each function calls the appropriate store's getState() method. For functions that need activeTabId (updatePolicy, setYaml, etc.), capture it from `usePolicyTabsStore.getState().activeTabId` at call time, plus the active tab's fileType from `usePolicyTabsStore.getState().getActiveTab()?.fileType`.
