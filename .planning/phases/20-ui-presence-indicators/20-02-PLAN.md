**Step 1: Create PresenceTabDots component**

Create `apps/workbench/src/features/presence/components/presence-tab-dots.tsx`:

1. Props interface:
   ```typescript
   interface PresenceTabDotsProps {
     /** The pane view route, e.g. "/file/{absolutePath}" */
     route: string;
   }
   ```

2. Imports:
   - `usePresenceStore` from `../stores/presence-store`
   - `usePaneStore` from `@/features/panes/pane-store`
   - `useCallback` from `react`

3. Component logic:
   - EARLY RETURN: If `!route.startsWith("/file/")`, return `null` — presence dots only apply to file tabs.
   - Extract file path from route: `const filePath = route.slice("/file/".length)`.
   - Read viewers for this file: `usePresenceStore((s) => s.viewersByFile.get(filePath))` — returns `Set<string> | undefined`.
   - Read local analyst ID: `usePresenceStore((s) => s.localAnalystId)`.
   - Read the full analysts Map: `usePresenceStore((s) => s.analysts)`.
   - IMPORTANT: Filter out the local analyst's fingerprint from the viewer set. Only show REMOTE viewers.
   - If no remote viewers (viewerSet is undefined/empty after filtering), return `null`.
   - Compute `remoteViewers`: array of `AnalystPresence` objects for up to 3 remote viewers.
   - Compute `overflow`: total remote viewer count minus 3 (if > 3).

4. Click handler — per locked decision "Clicking a presence dot navigates to that analyst's current file":
   ```typescript
   const handleDotClick = useCallback(
     (e: React.MouseEvent, analyst: AnalystPresence) => {
       e.stopPropagation(); // Prevent tab activation from the parent button
       if (!analyst.activeFile) return; // No-op when analyst has no active file
       const label = analyst.activeFile.split("/").pop() ?? analyst.activeFile;
       usePaneStore.getState().openFile(analyst.activeFile, label);
     },
     [],
   );
   ```

5. Render — each dot is a `<button>`, NOT a `<span>`, so it is interactive and accessible:
   ```tsx
   <span className="flex items-center gap-0.5 ml-1 shrink-0">
     {remoteViewers.slice(0, 3).map((analyst) => (
       <button
         key={analyst.fingerprint}
         type="button"
         className="inline-block w-[5px] h-[5px] rounded-full hover:ring-1 hover:ring-white/30 transition-shadow cursor-pointer"
         style={{ backgroundColor: analyst.color }}
         title={`${analyst.displayName} — click to open ${analyst.activeFile ?? "their file"}`}
         onClick={(e) => handleDotClick(e, analyst)}
       />
     ))}
     {overflow > 0 && (
       <span className="text-[8px] font-mono text-[#6f7f9a]/60 ml-0.5">
         +{overflow}
       </span>
     )}
   </span>
   ```
