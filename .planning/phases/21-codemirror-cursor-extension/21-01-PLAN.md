**1. RemoteCursorData interface and StateEffect:**
```typescript
interface RemoteCursorData {
  fingerprint: string;
  displayName: string;
  color: string;
  cursor: { line: number; ch: number } | null;
  selection: { anchorLine: number; anchorCh: number; headLine: number; headCh: number } | null;
}
```
- `export const updateRemoteCursors = StateEffect.define<RemoteCursorData[]>();`
- StateField that stores `RemoteCursorData[]`, updates on effect (same pattern as `guardRangesField` in guard-gutter.ts)

**2. CursorCaretWidget (extends WidgetType):**
- Constructor takes `displayName: string` and `color: string`
- `toDOM()` creates a `<span class="cm-remote-caret">` with:
  - `style.borderLeftColor = this.color` (2px wide, full line height)
  - Child `<span class="cm-remote-caret-label">` with `textContent = this.displayName`, `style.backgroundColor = this.color`, white text
  - Label hidden by default, shown on `:hover` via CSS (with 3s auto-fade via animation)
- `eq(other)` compares displayName and color
- `ignoreEvent() { return true; }` (don't capture clicks)

**3. ViewPlugin (presenceCursorsPlugin):**
Using `ViewPlugin.fromClass(class { ... }, { decorations: (v) => v.decorations })`:

- **Property `decorations: DecorationSet`** initialized in constructor
- **Property `unsubscribe: (() => void) | null`** for store subscription cleanup
- **Property `throttleTimer: ReturnType<typeof setTimeout> | null`** for outbound throttle
- **Property `lastSentSelection: string`** for dedup (JSON of last sent cursor/selection)
- **Property `currentFilePath: string | null`** tracks which file this editor is showing

**Constructor(view: EditorView):**
- `this.decorations = Decoration.none;`
- Subscribe to presence-store via raw `usePresenceStore.subscribe()` (NOT a React hook):
  ```typescript
  this.unsubscribe = usePresenceStore.subscribe((state) => {
    // Read remote cursors for current file, dispatch StateEffect
    const localId = state.localAnalystId;
    const cursors: RemoteCursorData[] = [];
    for (const [fp, analyst] of state.analysts) {
      if (fp === localId) continue; // Skip self
      if (analyst.activeFile !== this.currentFilePath) continue;
      if (!analyst.cursor) continue;
      cursors.push({
        fingerprint: fp,
        displayName: analyst.displayName,
        color: analyst.color,
        cursor: analyst.cursor,
        selection: analyst.selection,
      });
    }
    view.dispatch({ effects: updateRemoteCursors.of(cursors) });
  });
  ```

**update(update: ViewUpdate):**
- Rebuild decorations when the StateField changes (check `update.transactions` for the effect, or just rebuild on every update where the field value is read):
  ```typescript
  const cursors = update.state.field(remoteCursorsField);
  this.decorations = this.buildDecorations(update.view, cursors);
  ```
- If `update.selectionSet` (local selection changed), send outbound cursor update throttled to 50ms:
  ```typescript
  if (update.selectionSet) {
    this.scheduleOutboundCursor(update.view);
  }
  ```

**buildDecorations(view: EditorView, cursors: RemoteCursorData[]): DecorationSet:**
- For each cursor, convert line:column to absolute position using `view.state.doc.line(cursor.line).from + cursor.ch`
- Clamp line to `[1, view.state.doc.lines]` and ch to `[0, lineLength]` for safety
- Create `Decoration.widget({ widget: new CursorCaretWidget(name, color), side: 1 })` at cursor position
- If selection exists, convert anchor/head line:column to absolute positions, create `Decoration.mark({ class: "cm-remote-selection", attributes: { style: "background-color: ${color}33" } })` (33 = 20% opacity hex)
- Sort all decorations by position (required by RangeSet.of)
- Return `Decoration.none` if cursors is empty, otherwise `RangeSet.of(decorations, true)`
- Limit to first 20 cursors (MAX_REMOTE_CURSORS = 20 constant)

**scheduleOutboundCursor(view: EditorView):**
- If `this.throttleTimer` is set, return (throttle active)
- Set `this.throttleTimer = setTimeout(() => { this.throttleTimer = null; this.sendCursorUpdate(view); }, 50)`

**sendCursorUpdate(view: EditorView):**
- Read `view.state.selection.main` (head and anchor positions)
- Convert head absolute position to line:column via `view.state.doc.lineAt(head)` -> `{ line: lineObj.number, ch: head - lineObj.from }`
- If head === anchor (no selection), send: `getPresenceSocket()?.send({ type: "cursor", file_path: this.currentFilePath, line, ch })`
- If head !== anchor (selection), convert anchor similarly, send: `getPresenceSocket()?.send({ type: "selection", file_path: this.currentFilePath, anchor_line, anchor_ch, head_line, head_ch })`
- Dedup: stringify the message, compare to `this.lastSentSelection`, skip if identical
- `currentFilePath` is determined from the nearest React context -- but since this is outside React, it needs to be injected. **Use a Facet** for the file path:
  ```typescript
  export const presenceFilePath = Facet.define<string, string>({ combine: (v) => v[v.length - 1] ?? "" });
  ```
  Read in sendCursorUpdate as `view.state.facet(presenceFilePath)`

**destroy():**
- Call `this.unsubscribe?.()` to detach store listener
- Clear `this.throttleTimer` if set
- Set `this.decorations = Decoration.none`
