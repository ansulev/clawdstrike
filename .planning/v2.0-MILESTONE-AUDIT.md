# Milestone v2.0 Audit Report

## Summary

**Status:** gaps_found

## Root Cause

All 4 partial requirements share a single root cause: the client never sends `view_file` / `leave_file` messages to the server when files are opened or closed. The types exist in `types.ts`, the server fully handles them in `presence.rs`, but no client code triggers the send.

**Fix:** Add `view_file`/`leave_file` sends in `file-editor-shell.tsx` (or a pane-store subscription) when files are opened/closed. Approximately 20 lines of code.
