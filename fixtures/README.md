# Fixtures

Golden fixtures used by tests, parity checks, and deterministic validation.

Current fixture groups:

1. `fixtures/canonical/` - canonical payloads and expected encodings.
2. `fixtures/receipts/` - signed receipt vectors.
3. `fixtures/policy-events/` - policy/event simulation vectors.
4. `fixtures/certification/` - certification-related fixtures.
5. `fixtures/threat-intel/` - threat-intelligence fixture data.
