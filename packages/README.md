# Packages

Non-Rust packages grouped by product role.

Layout:

1. `packages/sdk/` - official SDKs (TypeScript and Python).
2. `packages/adapters/` - framework/runtime adapters.
3. `packages/policy/` - canonical TypeScript policy engine.

Ownership and maturity:

1. Owners are defined in `.github/CODEOWNERS`.
2. Maturity levels are defined in `docs/REPO_MAP.md`.
